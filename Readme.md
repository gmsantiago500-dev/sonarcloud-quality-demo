# SonarCloud Quality Demo

Proyecto simple en Python diseñado para ser analizado con SonarCloud como parte de la práctica de Calidad de Software.

Incluye:
- Bugs intencionales
- Code smells
- Vulnerabilidades
- Funciones simples para análisis

## Estructura
- main.py
- calculator.py
- utils.py
- data_processor.py

## Objetivo
Permitir el análisis completo en SonarCloud para comentar métricas como:
- Clean Code
- Software Quality
- Severity
- Type
- Technical Debt
